Number Guessing Game
Welcome to the Number Guessing Game repository! This simple game allows users to guess a randomly generated number within a specified range.

How to Play
copy the code from the source file and paste it on any java compiler.

Rules
The game will randomly generate a number within a specified range. You need to guess this number.
You'll be prompted to enter your guess.
If your guess is higher than the generated number, you'll be informed that your guess is too high.
If your guess is lower than the generated number, you'll be informed that your guess is too low.
If your guess matches the generated number, you'll be congratulated and the game will end.
You have a limited number of attempts to guess the correct number.
